    Funciones Anónimas

    const getName = () => "Manuel"

    console.log("Hola, " + getName);

    console.log((() => "Hola, Alfredo")()); Esto es una función anónima
    Se utiliza en casos que necesite una función puuntualmente y no reutilizable, o que sea autoejecutable. Para ahorrar tiempo y memoria no le asigno nombre, así tampooco se le puede llamar por "error", cuando solo neceesito que se ejecute una vez al cargar la página
    Una función anónima, se ejecutará solo si es autoejecutable o si la paso como parámetro.

    Funciones Flecha con return implicito:

    const resta = (a, b) => a - b;

    consoe.log(resta(5, 2));

    Funciones callback

    Es una función que se pasa como parámetro a otra función, para ser ejecutada después.
    function logCustom(content, log) {
    log(content)
    }

    logCustom("Omar", console.log)
    logCustom("Omar", logRojo)
    "logRojo" es también una función que me muestra el nombre en rojo. logCustom("Omar", logAzul)
